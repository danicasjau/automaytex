# AutoMayTex (In Development)

**AutoMayTex** is an automated, AI-driven PBR texture generation pipeline built directly for Autodesk Maya. It leverages Stable Diffusion (SDXL), ControlNet (Depth-Anything), and Arnold to extract geometry data, generate seamless textures based on text prompts, and automatically apply complete PBR materials back onto your 3D assets.

> **Work In Progress (In Development)**
> This project is currently in active development. Architecture, pipeline steps, and API are subject to change.
